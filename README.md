# ⌨️ Keylogger

This is a simple Python utility that records keystrokes to a local text file for educational and backup purposes.

---

## 📋 What is this project about?

The project uses the `pynput` library to capture input events from the keyboard and save them to a file named `keyFile.txt` in the same directory.

---

## 🚀 How to Run the Project

### 1. Install Dependencies
Before running the script, make sure you have the required `pynput` library installed:
```bash
pip install pynput
```

### 2. Run the Script
Execute the Python script in your terminal:
```bash
python keylogger.py
```

Once started, the program will monitor keystrokes and write standard characters to `keyFile.txt`. To stop the program, press Enter in the console window.
